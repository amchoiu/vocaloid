Project SEKAI COLORFUL STAGE! feat. Hatsune Miku

Project Sekai is a rhythm game released on September 30, 2020. This rhythm game is a collaboration between SEGA, Craft Egg, Colorful Palette, and Crypton Future Media. It is developed for Andriod and iOS, not like Project Diva the main character is not Hatsune Miku.

Their are six groups in this game: Virtual Singer, Leoneed, MoreMoreJump, VivdBadSquad, WonderlandxShowtime, and 25-ji Nightcord de.
