Hatsune Miku: Project DIVA F 2nd

Hatsune Miku: Project DIVA F 2nd is a rhythm game created by Sega and Crypton Future Media for the PlayStation in 2014.

The main character is Hatsune Miku. On top of that, Kagamine Rin, Kagamine Len, Megurine Luka, MEIKO and KAITO are added to Project Diva.
